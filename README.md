# Desktop-Games
Simple Windows Scripts that create games playable on your desktop!
They are made in Jscript, so that you can make them yourself easily, and they shouldn't get blocked/flagged by anything, this also helps because they can be made easily and without installing dependencies or special software.

# Steps:
1. Open notepad
2. Copy and paste on of the games into the file
3. Save it as (filename).js to your desktop
4. Drag the icons around to play games!
